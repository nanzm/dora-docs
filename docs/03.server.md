### 服务端
 - dora-server (本系统服务端的功能，使用 go 编写)

  接收到上报信息，过滤校验后，对消息体的部分属性进行`md5`，存入日志系统，为`web`控制台提供接口查询服务，同时包含了报警主要功能等，是监控系统的核心部分，采用`go`语言编写，追求稳定，高吞吐量

 - dora-sourcemap （接收 sourcemap 文件，并提供还原 error 堆栈信息的接口，使用 nodejs 编写）
