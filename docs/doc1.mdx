## 简述
Dora 是一个轻量、易部署、易使用、成本低的前端监控系统。
旨在帮组前端开发的兄弟们，把控`web`应用的稳定性，帮助开发者的随时查看应用上线情况，出现问题及时响应，提升安全感和幸福感。

## 工作原理

系统分为三个部分 sdk、server、web 控制台，sdk 采集上报数据，服务端接受到上报数据，进行校验过滤后，投递到消息中间件，随后进行消费存储到日志服务中。
而随后的分析，数据查询则依赖该[日志服务](https://help.aliyun.com/document_detail/48869.html)，系统设计简单可靠。

![系统流程图](/img/diagram-1.png)

### sdk

[https://www.npmjs.com/package/@dorasdk/js](https://www.npmjs.com/package/@dorasdk/js)

`sdk`实现轻量，使用了`typescript`编写，体积小，已在多个项目中使用和验证，
`gizp`后不到`10 kb`，优先使用了`sendBeacon`上报 不阻塞业务正常运行

### server
服务端接收上报信息，过滤和校验无效信息后，对消息体进行`md5`聚类，存入日志系统，为`web`控制台提供接口查询服务，同时包含了报警主要功能等，是监控系统的核心部分，采用`go`语言编写，追求稳定，高吞吐量


### web
通过大量的图表，展示重要指标数据，查看错误异常信息，堆栈信息 sourcemap 还原，支持成员管理，及一些配置管理功能。
